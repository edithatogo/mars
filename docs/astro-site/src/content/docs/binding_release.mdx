Binding releases follow the shared release inventory and the package-specific
paths in [Package Release Paths](package_release_paths).

The release process is intentionally conservative:

- stable runtime releases require artifact inspection and conformance checks
- training releases require the Rust training path to be ready
- publication uses protected environments and registry-specific approvals
