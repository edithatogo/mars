Portable `ModelSpec` artifacts are the shared runtime contract between
language bindings.

The current implementation supports:

- validation
- `design_matrix`
- `predict`
- export and import of portable specs

See [Rust Core](rust_core) and [Binding ABI and API Contract](binding_abi_contract)
for the runtime boundary.
