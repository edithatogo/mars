# Welcome to Mars

Mars is a **Go library and CLI tool** for analyzing structured time-series data. It provides statistical analysis, filtering, grouping, moving averages, time bucketing, and I/O utilities for CSV and JSON formats.

## Quick Start

```bash
# Install the CLI
go install github.com/edithatogo/mars/cmd/mars@latest

# Analyze a CSV file
mars --input data.csv --format table
```

## Features

## Project Structure

The project follows standard Go module layout:

- **`mars/`** — Core library package with all analysis logic
- **`cmd/mars/`** — CLI entry point
- **`docs/`** — Starlight-powered documentation site
- **`conductor/`** — Project planning and design artifacts
