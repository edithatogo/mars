# Changelog

## 1.0.0 (2025-01-01)

### Added
- Core MARS algorithm implementation
- Scikit-learn compatible interface with EarthRegressor and EarthClassifier
- Cross-validation support with EarthCV
- Feature importance calculations
- Plotting utilities for model diagnostics
- Support for generalized linear models (GLM) with GLMEarth
- Comprehensive test suite
- Documentation and examples
