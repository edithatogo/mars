# API Reference

The supported public Python API remains the documented `pymars` surface.

Stable estimators:

- `pymars.Earth`
- `pymars.EarthRegressor`
- `pymars.EarthClassifier`

Portable runtime helpers:

- `pymars.validate`
- `pymars.load_model_spec`
- `pymars.load_model`
- `pymars.save_model`
- `pymars.predict`
- `pymars.design_matrix`
- `pymars.inspect`
