# API Reference

Only the documented top-level `pymars` symbols below are part of the supported
public API. Undocumented imports and internal modules are not covered by
compatibility guarantees.

## Stable estimators

- `pymars.Earth`
- `pymars.EarthRegressor`
- `pymars.EarthClassifier`

## Portable runtime

- `pymars.validate`
- `pymars.load_model_spec`
- `pymars.load_model`
- `pymars.save_model`
- `pymars.predict`
- `pymars.design_matrix`
- `pymars.inspect`

## Experimental APIs

- `pymars.GLMEarth`
- `pymars.EarthCV`
